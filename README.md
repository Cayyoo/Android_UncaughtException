# UncaughtException
自定义全局异常捕获

```java
/**
 * 自定义处理异常：
 * 全局异常捕获
 *
 * 1、自定义异常处理类CrashHandler并实现接口Thread.UncaughtExceptionHandler
 * 2、自定义CrashApp继承Application
 * 3、<application/>标签中给出android:name=".CrashApp"
 */
```
